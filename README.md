# Digital-Clinic-Book
This project was developed to address the issue of having to use traditional clinic books in Papua New Gunea. The primary aim is to eliminate the reliance on traditional books and move to a digital format.

## What is the a Digital Clinic Book?
This offers a digital format of the traditional clinic book. It allows users to freely access their full medical records when going to hospitals for treatment. These records would be stored online and be accessed by registered medical professionals only after going through a vetting process.

## What it provides.
It provides a plateform to builde of where medical professional can access medical records of patients that have prior treatment from other hospitals. This can help speed up diagnosis and treatment.

This repository serves as a demonstration of what this project would look like fully developed.
